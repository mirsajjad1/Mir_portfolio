# Mir Mohammad Sajjadul Islam | Portfolio

A clean and responsive personal portfolio website showcasing my skills, projects, and contact information. Built using **HTML**, **CSS**, **Bootstrap 5**, **JavaScript**, and **AOS animation library**. This project demonstrates my focus on **backend development with Django** while providing a polished frontend for personal branding.

---

## Features

- **Responsive Navbar:** Fixed-top navigation with smooth scrolling between sections.
- **Hero Section:** Eye-catching introduction with profile image, name, and role description.
- **About Section:** Overview of personal skills, goals, and learning focus.
- **Skills Section:** Highlights technical skills: Python, Django, HTML, CSS, SQLite.
- **Projects Section:** Showcases mini projects like:
  - Mini Twitter Clone
  - Doctor Appointment System
- **Contact Section:** Quick email contact button.
- **Scroll Progress Bar:** Visual indicator of page scroll progress.
- **Animations:** Smooth fade and slide animations via [AOS](https://michalsnik.github.io/aos/).

---

## Technologies Used

- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript
- **Animations:** AOS (Animate on Scroll)
- **Backend Skills Demonstrated:** Django, SQLite (projects linked)
- **Version Control:** Git & GitHub
